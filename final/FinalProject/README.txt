This program was created for my Programming With Classes final project.

This program is a Vehicle Manager Software.
It will create a new vehicle out of a selection of 8 different vehicle.
It can also refuel, start engine, save to file, load from file, and check the stats on a certain vehicle.

To use this program, compile it and start up the program.
Press 0-6 to select what option you want.
Everything else is explained as you go through the program.
